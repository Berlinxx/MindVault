## ? Quick Commands

### Build Package
```cmd
BUILD_MSIX.bat
```

### Test Installation (on another PC)
```cmd
1. Right-click run_me_first.bat ? Run as admin
2. Double-click MindVault.msix
```

### Check Windows Version
```cmd
winver
```
Need: Windows 10 version 2004 or later

### Check .NET Version
```cmd
dotnet --version
```
Need: 9.0.x

## ?? Key Points

### ? DO:
- ? Run BUILD_MSIX.bat from project root
- ? Share .msix + .cer + run_me_first.bat
- ? Keep .pfx file secure
- ? Use same .pfx for updates
- ? Test on another PC first

### ? DON'T:
- ? Share the .pfx file
- ? Delete the .pfx (you need it for updates!)
- ? Skip certificate installation
- ? Forget to run build_msix.bat as admin (not needed)
- ? Build while app is running

## ?? Troubleshooting Quick Guide

| Issue | Solution |
|-------|----------|
| "makeappx not found" | Install Windows SDK |
| "Build failed" | Run from project root |
| "Certificate error" | User: Run run_me_first.bat as admin |
| "Can't run on PC" | Need Windows 10 v2004+ |
| "Install failed" | Uninstall old version first |
