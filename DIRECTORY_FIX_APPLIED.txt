                     HOW IT WORKS
???????????????????????????????????????????????????????????????????????????????

Batch File (.bat):
  %~dp0 = Drive and Path of the batch file (parameter 0)
  cd /d "%~dp0" = Change directory to where the batch file is

PowerShell (.ps1):
  $PSScriptRoot = Directory containing the PowerShell script
  Set-Location -Path $PSScriptRoot = Change to script directory
