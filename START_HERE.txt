?? TROUBLESHOOTING QUICK REFERENCE
???????????????????????????????????????????????????????????????????????????????

Problem: "Cannot install package"
Solution: Install certificate first
    certutil -addstore TrustedPeople "MindVault_Certificate.cer"

Problem: "python311.dll not found"
Solution: Delete Python folder and restart app
    Remove-Item "$env:LOCALAPPDATA\MindVault\Python311" -Recurse -Force

Problem: App won't launch
Solution: Reinstall
    Get-AppxPackage -Name "*mindvault*" | Remove-AppxPackage
    # Then run INSTALL.bat again

Problem: Old version won't uninstall
Solution: Force remove
    Get-AppxPackage -AllUsers -Name "*mindvault*" | Remove-AppxPackage -AllUsers
